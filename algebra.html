Algebra: The Science of Generalization

At its heart, algebra is the powerful art of using symbols to generalize rules and relationships. This page covers the foundational concepts of solving for unknowns and understanding the relationships between variables.

Part 1: Equations & Inequalities

This section focuses on the toolkit for solving for unknown values. An equation is like a balanced scale, and our goal is to find the value that keeps it in balance.

The Foundation: What is an Equation?

An equation states that two expressions are equal (e.g., $x+3=5$). Our goal is to find the value of the variable (the symbol, like $x$) that makes this statement true. We do this by keeping the "scale" balanced.

One-Step & Two-Step Equations

We solve equations using inverse operations to isolate the variable. Addition undoes subtraction, and multiplication undoes division. For two-step equations, we reverse the order of operations (undoing addition/subtraction first).

Example (Two-Step): Solve $3x - 4 = 11$.
1. Add 4 to both sides: $3x = 15$.
2. Divide both sides by 3: $x = 5$.

Multi-Step & Special Cases

For more complex equations, we first simplify by combining like terms or using the distributive property to remove parentheses. Sometimes, the variables will cancel out, leading to a true statement like $5=5$ (Infinite Solutions) or a false statement like $5=2$ (No Solution).

Inequalities

Inequalities (using $<, >$) are solved just like equations, with one golden rule: when you multiply or divide both sides by a negative number, you must flip the inequality sign.

Example: Solve $-2x > 10$.
Divide by -2 and flip the sign: $x < -5$.

Part 2: Functions

Functions describe a specific relationship between an input and an output. For every one input, there is exactly one output.

What is a Function?

Think of a function as a machine. You put an input (like a number) in, the machine follows a rule, and it produces a single output. We can use the Vertical Line Test on a graph to see if it's a function: if a vertical line touches the graph more than once, it's not a function.

Function Notation

We use the notation $f(x)$ (read "f of x") to represent a function. This is a replacement for 'y' that makes the input-output relationship clear.

Example: If $f(x) = x^2 + 2$, find $f(3)$.
Replace x with 3: $f(3) = (3)^2 + 2 = 9 + 2 = 11$.
