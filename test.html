Question 1: What is the value of $x$ in the equation $2x + 3 = 11$?

Answer: $x = 4$. To solve, subtract 3 from both sides to get $2x = 8$, then divide by 2.

Question 2: What does the term 'function' mean in algebra?

Answer: A function is a rule that assigns each input value exactly one output value.
