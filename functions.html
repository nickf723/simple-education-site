The Foundation: What is a Function?

A function is one of the most important ideas in mathematics. Think of it as a machine that follows a specific rule. You give it an input, the machine does something to it, and it gives you a predictable output.

The Key Rule: For a relationship to be a function, every single input must have exactly one output. An input can't give you two different outputs!

The "Function Machine"

The Vertical Line Test

A quick way to tell if a graph represents a function is the Vertical Line Test. If you can draw a vertical line anywhere on the graph that crosses the line or curve more than once, it is not a function.

Essential Vocabulary

  • Relation: Any set of input and output pairs.
  • Function: A special relation where each input has only one output.
  • Domain: The set of all possible inputs (x-values) for a function.
  • Range: The set of all possible outputs (y-values) for a function.

Check Your Understanding

Question 1: An input of 3 gives an output of 6. An input of 3 also gives an output of 7. Is this a function?

Answer: No. The input 3 has two different outputs (6 and 7), which violates the key rule of functions.

Question 2: If you draw a circle on a graph, would it pass the Vertical Line Test?

Answer: No. A vertical line would cross the circle in two places, so a circle is not a function.
